Components 核心概念

1. 组件系统概述

GrapesJS 中的组件是构建页面的基本单元，每个HTML元素都对应一个组件实例。

editor.Components.getWrapper()

2. 内置组件类型

• default: 默认组件

• text: 文本组件

• image: 图片组件

• link: 链接组件

• table: 表格组件

type: 'text', content: 'Hello'

3. 组件识别

通过 isComponent 方法识别HTML元素应该使用哪种组件类型。

isComponent: (el) => el.tagName === 'IMG'

4. 自定义组件

可以定义自定义组件类型，扩展默认行为。

editor.Components.addType('my-component', {...})
