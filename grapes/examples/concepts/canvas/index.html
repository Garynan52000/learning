Canvas 核心概念

1. 画布概述

Canvas 是 GrapesJS 的编辑区域，提供可视化的页面编辑环境。

editor.Canvas.setZoom(50)

2. 缩放功能

支持画布缩放，方便在不同尺寸下编辑内容。

• 缩放范围: 10% - 200%

• 平滑缩放动画

canvas.getZoom() // 获取当前缩放

3. 坐标系统

画布使用坐标系统来定位元素和处理拖拽操作。

canvas.setCoords(100, 100)

4. 拖拽系统

支持组件的拖拽操作，包括自定义拖拽源。

canvas.startDrag({content: '...'})

5. 画布事件

监听各种画布事件，如拖拽、缩放、坐标变化等。

editor.on('canvas:zoom', () => {...})
