Blocks 核心概念

1. 块管理器概述

Block Manager 管理可重用的内容块，用户可以拖拽这些块到画布中创建组件。

editor.BlockManager.add('my-block', {...})

2. 块的内容类型

• 组件导向: 使用组件定义

• HTML字符串: 直接HTML内容

• 混合模式: 组件+样式

content: { type: 'text', content: 'Hello' }

3. 块的属性

• id: 唯一标识符

• label: 显示名称

• content: 块内容

• category: 分类

• attributes: HTML属性

category: 'Basic', media: '📝'

4. 程序化使用

可以通过API动态添加、删除和管理块。

blockManager.get('my-block')
